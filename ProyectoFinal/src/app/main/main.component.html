¡Bienvenido a la mejor Web de Heroes de Marvel del mundo!


La Web de Marvel Comics es una herramienta para ayudar a la gente que quiere conocer sobre sus Heroes de Marvel, esta web utiliza datos de más de 70 años de la era de los cómics de Marvel.


Finalmente, recuerde que el portal y la suite de API de Marvel están en versión beta y este es un proyecto en evolución. En la medida de lo posible, intentaremos comunicar los cambios en la API antes de que ocurran e intentaremos limitar cualquier modificación que provoque cambios incompatibles con las versiones anteriores de las aplicaciones.
